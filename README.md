# Archive - wjdrkdnj_.s

Photography portfolio website showcasing a curated collection of images.

## Features

- Responsive 3-column grid layout
- Image modal with EXIF data display
- Keyboard navigation support
- Lazy loading for optimal performance
- Minimal, museum-like aesthetic

## Technologies

- HTML5
- CSS3
- Vanilla JavaScript
- EXIF.js for metadata extraction

## Deployment

This site can be deployed to:
- Netlify (recommended)
- Vercel
- GitHub Pages
